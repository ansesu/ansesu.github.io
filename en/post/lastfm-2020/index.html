My 2020 last.fm

Andre
Jan 2, 2021 3 min read

last.fm is a social network in which users can keep track of their own musical activity through the so called scrobbles (the log of your listened song). This service was created in 2002, when it was necessary to install a software in order to scrobble. Recently, however, the plataform was integrated to music streaming services such as Spotify, Deezer, Tidal and Google Play Music. Thus, it has become much easier to keep track of your own musical activity. last.fm shows weekly (in the free version), monthly and annual (in the premium version) statistics, concerning the number of listened songs, artists, albums, genres, temporal patterns and comparison with other last.fm users. In addition, the plataform allows users to find people with similar musical tastes.

Here, I perform a statistical analysis of my last.fm profile in 2020. I start by using the Last.fm data export tool to extract my profile’s listening history. I use the Python to parse data and the package d3.js to make some visualizations. My code is available at: https://github.com/ansesu/lastfm/.
