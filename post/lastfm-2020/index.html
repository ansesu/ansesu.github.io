Meu last.fm em 2020

Andre
Jan 2, 2021 3 minutos de leitura

O last.fm é uma rede social em que os usuários conseguem manter um histórico de sua atividade musical por meio de scrobbles (nome atribuído ao registro de cada música reproduzida pelo usuário). Esse serviço existe desde 2002, quando a instalação de um programa era necessária para realizar os scrobbles. Recentemente, porém, a plataforma foi integrada a vários serviços de streaming como Spotify, Deezer, Tidal e Google Play Music. Sendo assim, ficou muito mais fácil manter o registro de sua própria atividade musical. O last.fm exibe estatísticas semanais (na versão gratuita), mensais e anuais (na versão paga), referentes ao volume de músicas escutadas, artistas, álbuns, gêneros, padrões temporais e comparações com outros usuários. Além disso, a plataforma permite os usuários encontrarem pessoas com gostos musicais similares ao seus.

Este ano, novamente vou realizar uma análise estatística do meu perfil no last.fm em 2020. Para isso, utilizei a ferramenta Last.fm data export para exportar o histórico do meu perfil. A partir daí, utilizei o Python para tratar o dado, e o pacote d3.js para confeccionar algumas visualizações. O código para reprodução está disponível em: https://github.com/ansesu/lastfm/.
