Visualização de dados com d3.js

Andre
Jan 2, 2021 2 minutos de leitura

Apesar de a confecção de visualizações diretamente pelo Python (ou qualquer outra linguagem de programação) ser mais prática, perde-se liberdade na personalização de interatividade e também responsividade ao expor essas imagens em formato da web. Por isso, resolvi aprender um pouquinho sobre a biblioteca d3.js. Abaixo, mostro algumas visualizações que confeccionei (as figuras são interativas).

Figura: gráfico de linha com barra vertical interativa indicando o valor da variável.

Figura: gráfico de linha com intervalo de confiança e com barra vertical interativa indicando o valor da variável.

Figura: gráfico multilinhas com barra vertical interativa indicando o valor das variáveis.

Figura: gráfico de linha com intervalo de confiança e cores diferentes definidas por um limite. Valores acima do limite são coloridos de vermelho, enquanto valores abaixo desse mesmo limite são coloridos de verde. Visualização inspirada no site rt.live.

Figura: scatter plot com tamanho do marker variável de acordo com valor da variável (com botão para troca de categoria).

Figura: gráfico de barras simples.

Figura: gráfico de barras agrupado (com botão para troca de categoria).

Figura: gráfico de barras radial.

Figura: stream chart plot.

Figura: gráfico de corrida de barras.
